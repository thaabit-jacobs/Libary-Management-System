Libary management system spec:

  • The project is specifically designed for for the use of libarians and libary users
  • The product will work as a complete user interface for libary management process and libary usage from ordinary users
  • Users: libarians libary users
  • The system will provide a search functionality to facilate the search of resouces.The search will be based on various categories bookname or the ISBN.
  • The libary staff personnel can add/update the resouces and the resouce users from the system.The users of the system can request issue/renew/return of books
  • Provide inforamtion on availible books in the libary
  • Provide information about the user
  • The libarian will act as as the adminstartor to control members and manage vooks
